How to play

  • Collect bubbles (small circles) to fill up your oxygen
  • Avoid other live species, else die
  • Collect power-ups (large circles) to be able to eat other species
  • Collect coins and other valuable items to buy upgrades

Upgrades

  • Pull bubbles
  • Move around faster
  • Push enemies
  • Longer oxygen usage
  • Longer power-up
  • Reduce game speed

Collectible items

  • Coins: 1-100 coins
  • Small diamond: 1k coins, distance 5k+
  • Large diamond: 10k coins, distance 7k+
  • Key: 100k coins, distance 10k+
  • Crown: 1M coins, distance 15k+

Ocean is an open-source game made with Polymatic framework. Check out the project github repository for more information.
